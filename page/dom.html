Calculadora DOM

Esta aplicación es una calculadora completamente funcional desarrollada utilizando JavaScript. Se ejecuta en el navegador web y se centra fundamentalmente en la manipulación del DOM (Document Object Model) y la programación orientada a eventos para ofrecer una experiencia de usuario interactiva y fluida. La aplicación se basa en la respuesta dinámica a las acciones del usuario, sin depender de recargas de página.
